# Food Recipe Web App

This documentation describes the design, functionality, and business purpose of the Food Recipe Web App.

The project is a full-stack web application built using Node.js, Express, React, and MongoDB.  
It enables users to create, edit, and manage cooking recipes with image uploads.

## Documentation Overview
- [Functional Document](functional_doc.md)

## Objective
To build a user-friendly web application that allows secure recipe management and image uploads using a scalable backend and responsive frontend.
